# Source Control

## GIT

```
git clone XXXXXXXXX
git init
```

```
git status
git add .
git commit -am "{MESSAGE}"
git reset --hard
git clean -f -d
```

```
git branch
git checkout [-b] {BRANCH}
git merge {BRANCH}
```

```
git pull origin {BRANCH}
git push origin {BRANCH}
git mergetool
```

### Special Files

- .git - folder holding the git database
- .gitignore - special file to represent wildcard matches to ignore
